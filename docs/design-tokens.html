Accent

The design-token foundation

Every visual constant in the product, named the way the code names it. Not components — the raw layer beneath them: color, radius, spacing, type, and the vocabulary to reference them precisely.

Two orthogonal axes compose freely: Mode (light / dark — neutral surfaces) and Accent (the primary color only). Flip either control above and every swatch below re-resolves live, printing the real authored value.

Naming rule. Each token is one CSS variable --x exposed as Tailwind utilities bg-x / text-x / border-x / ring-x. When talking to an AI agent, name the token (bg-primary, text-muted-foreground) — never a hex. Colors are authored in oklch.
Theming model

Two axes, set on the <html> element

Mode and accent set disjoint variables, so cascade order between them never matters. Both are replayed before first paint by a boot script and persisted to localStorage.
